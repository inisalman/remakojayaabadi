# Homepage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build responsive static Nuxt homepage for PT. Remako Jaya Abadi with exact approved Indonesian copy and JSON mock data.

**Architecture:** Global layout owns header/footer/main landmark. Homepage imports CMS-shaped JSON records and renders shared components; selected project category is its only local state. Client plugin adds reveal state while reduced-motion users see content without animation.

**Tech Stack:** Nuxt 3, Vue 3 `<script setup lang="ts">`, TypeScript, Tailwind CSS, `@nuxtjs/google-fonts`, `lucide-vue-next`, Vitest, Vue Test Utils.

## Global Constraints

- Exact homepage copy from `docs/page-copy.md`; no substitute copy.
- All mock content lives in `frontend/data/*.json`.
- Routes: `/tentang-kami`, `/layanan`, `/proyek`, `/kontak`.
- Tailwind tokens only in templates; no literal color hex.
- Inter: weights `400`, `500`, `600`, `700` via `@nuxtjs/google-fonts`.
- Import Lucide icons individually.
- Mobile-first; semantic landmarks, skip link, focus styles, alt text, accessible menu.
- `v-reveal` uses `IntersectionObserver`; reduced-motion immediately reveals.
- Below-fold images lazy-load. Hero image URL: `https://placehold.co/1200x675/4647AE/FFFFFF?text=Construction`.
- `npm run generate` must pass.
